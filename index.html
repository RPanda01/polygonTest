Управление полигонами

Инструкции:

  • Нажмите "Создать полигоны" для генерации случайных полигонов
  • Перетаскивание: Зажмите полигон в буферной зоне и перетащите в рабочую зону
  • Альтернатива: Двойной клик по полигону в буферной зоне для копирования
  • Рабочая зона: колесико мыши для масштабирования, перетаскивание фона для панорамирования
  • Удаление: Двойной клик по полигону в рабочей зоне
  • "Сохранить" - сохранить текущие полигоны, "Сбросить" - удалить сохраненные данные
